# 한국 고대사 키워드 정리 ⚡

## 🔥 핵심 연표 (시간순 정확)
- **371년** 근초고왕 → 고국원왕 전사 (평양성)
- **372년** 소수림왕 → 불교수용(순도), 태학설립
- **373년** 소수림왕 → 율령반포
- **399년** 광개토왕 → 신라 구원, 금관가야 쇠퇴
- **427년** 장수왕 → 평양천도, 남진정책
- **433년** 나제동맹 성립 (비유왕-눌지왕)
- **475년** 장수왕 → 개로왕 전사, 문주왕 웅진천도
- **493년** 동성왕-소지왕 혼인동맹
- **532년** 법흥왕 → 금관가야 병합
- **551년** 나제동맹 → 한강유역 탈환
- **553년** 신라 배신 → 한강독점, 신주설치
- **554년** 관산성 전투 → 성왕 전사 (김무력이 죽임)
- **562년** 진흥왕 → 대가야 멸망 (이사부, 사다함)

## 🏰 고구려
### 주요 왕 (시간순)
- **고국천왕** - 을파소 등용, 진대법
- **고국원왕** - 근초고왕에게 전사 (371년)
- **소수림왕** - 불교수용(372), 태학설립(372), 율령반포(373)
- **광개토왕** - 영락연호, 신라구원(399), 금관가야 쇠퇴시킴
- **장수왕** - 평양천도(427), 남진정책, 개로왕 죽임(475)
- **영양왕** - 살수대첩 (을지문덕), 신집편찬 (이문진)
- **영류왕** - 대당정책 온건파
- **연개소문** - 영류왕 죽이고 보장왕 세움, 천리장성 축조

### 주요 사건
- **도읍이동** 졸본 → 국내성 → 평양
- **안시성 전투** - 고구려가 당나라 막아냄
- **관구검** - 동천왕과 전쟁, 환도성 함락

## 🏯 백제 (정확한 연대)
### 주요 왕 (시간순)
- **근초고왕** - 서기편찬(고흥), 고국원왕 전사시킴(371)
- **개로왕** - 북위에 사신파견, 장수왕에게 전사(475)
- **문주왕** - 웅진(공주) 천도(475)
- **동성왕** - 나제동맹 강화, 혼인동맹(493)
- **무령왕(501~523)** - 22담로, 중국남조 교류, 벽돌무덤
- **성왕(523~554)** - 사비천도, 남부여 국호, 관산성에서 전사
- **무왕** - 금마저 천도시도, 미륵사지 석탑
- **의자왕** - 대야성 점령 (김춘추 딸 죽음)

### 주요 사건
- **천도과정** 한성(위례성) → 웅진(공주) → 사비(부여)
- **관산성 전투(554)** - 성왕이 김무력(금관가야 출신)에게 전사
- **백제부흥운동** - 복심·도침이 부여풍을 왕으로 세움

## 🏛️ 신라
### 주요 왕 (시간순)
- **내물왕** - 김씨 세습 확립
- **눌지왕** - 나제동맹 체결(433)
- **소지왕** - 동성왕과 혼인동맹(493)
- **지증왕** - 순장금지, 이사부로 우산국 정벌
- **법흥왕** - 이차돈 순교, 불교공인, 건원연호, 금관가야 병합(532)
- **진흥왕** - 한강독점(553), 대가야 멸망(562), 화랑도 개편, 거칠부에게 국사편찬, 순수비
- **무열왕** - 강수가 청방인문표 작성
- **문무왕** - 외사정 파견, 감은사
- **신문왕** - 김흠돌의 난 진압, 국학설치, 9주정비, 관료전 지급, 녹읍폐지, 9서당·10정, 만파식적

### 주요 인물
- **이사부** - 우산국 정벌, 대가야 멸망
- **김무력** - 성왕 죽인 금관가야 출신, 김유신 할아버지
- **거칠부** - 진흥왕때 국사편찬
- **원광법사** - 세속5계 제시
- **설총** - 원효 아들, 이두정리, 화왕계
- **최치원** - 6두품, 시무10조, 황소의난 격황소서

## ⚔️ 가야 (정확한 시기)
### 금관가야 (김해)
- **김수로왕** 전설적 시조
- **399년** 광개토왕 공격으로 쇠퇴 시작
- **532년** 법흥왕에게 병합

### 대가야 (고령) 
- 5세기 후반 금관가야 쇠퇴 후 가야연맹 중심
- **우륵** - 가야금, 신라 투항
- **562년** 진흥왕(이사부·사다함)에게 멸망

## 🌊 발해
### 연호
- **무왕** - 인안, 등주 선제공격
- **문왕** - 대흥
- **선왕** - 건흥

## 📜 중요 연호
- **광개토왕** - 영락
- **법흥왕** - 건원  
- **발해 무왕** - 인안
- **발해 문왕** - 대흥
- **발해 선왕** - 건흥
- **고려 태조** - 천수
- **고려 광종** - 광덕, 준풍
- **조선 고종** - 개국, 건양
- **대한제국 고종** - 광무
- **대한제국 순종** - 융희

## 🎨 문화·제도
### 정치제도
- **고구려** - 제가회의, 욕살·처려근지, 태학·경당
- **백제** - 정사암회의, 부여씨·8성귀족, 16관등제
- **신라** - 화백회의, 골품제, 위화부·영객부

### 교육·문화
- **향가집** - 삼대목 (위홍·대구화상)
- **이두** - 설총이 정리
- **도교영향** - 백제 금동대향로, 고구려 사신도

### 무덤양식
- **고구려** - 돌무지무덤, 고분벽화 (장군총, 덕흥리 등)
- **백제** - 능산리고분군(부여), 송산리고분군(공주), 무령왕릉(벽돌무덤)
- **신라** - 돌무지덧널무덤, 황금문화 (천마총, 금관 등)

## ⚡ 시험 핵심 암기
### 연대 필수 암기
- **372년** - 소수림왕 불교수용·태학설립
- **373년** - 소수림왕 율령반포
- **433년** - 나제동맹 성립
- **475년** - 개로왕 전사, 웅진천도  
- **532년** - 금관가야 멸망
- **554년** - 관산성 전투, 성왕 전사
- **562년** - 대가야 멸망

### 헷갈리기 쉬운 구분
1. **무령왕(501~523) vs 무열왕(654~661)** 
   - 무령왕 = 백제, 22담로, 벽돌무덤
   - 무열왕 = 신라, 김춘추, 백제 멸망
2. **금관가야(532) vs 대가야(562)**
   - 금관가야 = 김해, 법흥왕이 멸망
   - 대가야 = 고령, 진흥왕이 멸망
3. **나제동맹 과정**
   - 433년 성립 → 493년 혼인동맹 → 553년 배신 → 554년 완전파탄
