# 한국 고대사 키워드 정리 ⚡ (완전판)

## 🔥 핵심 연표 (시간순 정확)
- **371년** 근초고왕 → 고국원왕 전사 (평양성), 서기편찬(고흥)
- **372년** 소수림왕 → 불교수용(순도), 태학설립
- **373년** 소수림왕 → 율령반포
- **399년** 광개토왕 → 신라 구원, 금관가야 쇠퇴
- **427년** 장수왕 → 평양천도, 남진정책
- **433년** 나제동맹 성립 (비유왕-눌지왕)
- **475년** 장수왕 → 개로왕 전사, 문주왕 웅진천도
- **493년** 동성왕-소지왕 혼인동맹 (이벌찬 비지의 딸)
- **532년** 법흥왕 → 금관가야 병합
- **551년** 나제동맹 → 한강유역 탈환
- **553년** 신라 배신 → 한강독점, 신주설치
- **554년** 관산성 전투 → 성왕 전사 (김무력이 죽임)
- **562년** 진흥왕 → 대가야 멸망 (이사부, 사다함)

## 🏰 고구려 (완전판)
### 주요 왕 (시간순)
- **미천왕** - 서안평 공격, 낙랑군 축출
- **고국천왕** - 을파소 등용, 진대법 시행
- **고국원왕** - 근초고왕에게 전사 (371년)
- **소수림왕** - 불교수용(372, 순도), 태학설립(372), 율령반포(373)
- **광개토왕** - 영락연호, 신라구원(399), 금관가야 쇠퇴시킴, 관미성 탈취, 후연 공격, 왕자 거련을 태자로 삼음
- **장수왕** - 평양천도(427), 남진정책, 개로왕 죽임(475), 신라 실직주성 탈취, 북위에 사신파견
- **동천왕** - 관구검과 전쟁, 환도성 함락
- **영양왕** - 살수대첩 (을지문덕), 신집편찬 (이민진이 유기 간추림)
- **영류왕** - 대당정책 온건파
- **연개소문** - 영류왕 죽이고 보장왕 세움(막리지의 난), 천리장성 축조

### 주요 사건
- **도읍이동** 졸본(최초) → 국내성(초기) → 평양(마지막)
- **안시성 전투** - 고구려가 당나라 막아냄
- **고구려 부흥운동** - 검모잠이 안승을 왕으로 추대, 보덕국왕

### 문화/제도
- **제가회의, 욕살·처려근지, 태학·경당**
- **고분벽화** - 장군총(돌무지무덤), 덕흥리고분(북두칠성벽화), 수렵도·씨름도·사신도
- **금동연가 7년명 여래입상** - 경남 의령군

## 🏯 백제 (완전판)
### 주요 왕 (시간순)
- **온조** - 서울에 나라 세움
- **근초고왕** - 서기편찬(고흥), 고국원왕 전사시킴(371), 평양성 공격
- **개로왕** - 북위에 사신파견, 장수왕에게 전사(475)
- **문주왕** - 웅진(공주) 천도(475)
- **동성왕** - 나제동맹 강화, 혼인동맹(493)
- **무령왕(501~523)** - 22담로, 중국남조 교류, 벽돌무덤, 매지권
- **성왕(523~554)** - 사비천도, 남부여 국호, 중앙 22부로 개편, 관산성에서 전사
- **무왕** - 의자왕 아버지, 금마저(익산) 천도시도, 미륵사 창건, 서동설화
- **의자왕** - 대야성 점령 (김춘추 딸 죽음)

### 주요 사건
- **천도과정** 한성(위례성) → 웅진(공주) → 사비(부여)
- **관산성 전투(554)** - 성왕이 김무력(금관가야 출신)에게 전사
- **백제부흥운동** - 복심·도침이 부여풍을 왕으로 세움

### 문화/제도/유적
- **정사암회의, 부여씨·8성귀족, 16관등제**
- **고분군** - 능산리고분군(부여/사비), 송산리고분군(공주/웅진)
- **공산성** = 웅진성
- **왕궁리 유적** - 익산, "마수부" 글자 기와 출토, 오층석탑
- **금동대향로** - 부여(사비) 출토, 도교사상 영향
- **산수무늬 벽돌** - 도교사상 영향

## 🏛️ 신라 (완전판)
### 주요 왕 (시간순)
- **내물 마립간** - 김씨 세습 확립
- **눌지왕** - 나제동맹 체결(433)
- **소지왕** - 동성왕과 혼인동맹(493)
- **지증왕** - 순장금지, 이사부로 우산국 정벌, **'왕' 호칭 최초 사용(503년)**, 국호 '신라' 확정
- **법흥왕** - 이차돈 순교, 불교공인, 건원연호, 병부·상대등 설치, 금관가야 병합(532)
- **진흥왕** - 한강독점(553), 대가야 멸망(562), 화랑도 개편, 거칠부에게 국사편찬, 순수비(창녕비), 황룡
- **진평왕** - 원광법사 세속5계
- **선덕왕** - 예성강 이북 패강진 설치
- **무열왕** - 강수가 청방인문표 작성
- **문무왕** - 외사정 파견, 감은사
- **신문왕** - 김흠돌의 난 진압, 국학설치, 9주5소경 정비, 관료전 지급, 녹읍폐지, 9서당·10정, 만파식적, 감은사지3층석탑, 초계문신제, 병부
- **원성왕** - 독서삼품과
- **진성여왕** - 최치원 시무10조

### 주요 인물
- **이사부** - 우산국 정벌, 금관가야·대가야 복속, 한강 차지에 중요 인물
- **김무력** - 성왕 죽인 금관가야 출신, 김유신 할아버지
- **거칠부** - 진흥왕때 국사편찬, 대아찬
- **원광법사** - 세속5계 제시, 화랑도 규범
- **설총** - 원효 아들, 이두정리, 화왕계를 신문왕에게 바침
- **최치원** - 6두품, 시무10조, 황소의난 격황소서, 당나라 유학·빈공과 급제
- **강수** - 무열왕때 문장가, 청방인문표 작성
- **설계두** - 6두품 출신 장수, 당나라 귀화
- **장보고** - 청해진
- **혜초** - 왕오천축국전

### 문화/제도/유적
- **화백회의, 골품제, 위화부·영객부**
- **화랑도** - 신라 초기 원화에 기원
- **돌무지덧널무덤** - 천마총(천마도), 신라금관
- **삼대목** - 향가모음집 (위홍·대구화상 편찬)
- **도선국사비, 무학대사비** - 김정희가 금석과안록에서 진흥왕 순수비 입증

## ⚔️ 가야 (완전판)

- **김수로왕** 전설적 시조
- **구지봉** 전설

- **399년** 광개토왕 공격으로 쇠퇴 시작

### 대가야 (고령)

- **우륵** - 가야금, 신라 투항
- **562년** 진흥왕(이사부·사다함)에게 멸망
- **창녕비** - 대가야 멸망 후 진흥왕이 세운 순수비

## ⚡ 주요 전투와 사건

- **살수대첩** - 을지문덕이 수나라 격파

## 🔑 시험 핵심 암기
### 절대 틀리면 안 되는 것들
1. **서기 편찬 = 무조건 근초고왕**
2. **22담로 = 무령왕 / 22부 = 성왕**
3. **벽돌무덤 = 무령왕릉**
4. **금관가야(532) vs 대가야(562)**
5. **능산리(부여) vs 송산리(공주)**
6. **욕살·처려근지 = 무조건 고구려**
7. **부여씨·8성귀족 = 백제**
