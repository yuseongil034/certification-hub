## ⚔️ 삼국통일 과정 (상세)

### 백제 멸망과 부흥운동
- **대야성 전투** - 의자왕이 김춘추 딸과 사위 공격 → 김춘추 당나라行
- **황산벌 전투** - **계백** 장군이 김유신에게 패배, 나당연합에 죽음
- **사비성 함락** - 나당연합으로 백제 멸망
- **백제부흥운동** - 복신·도침이 부여풍을 왕으로 추대 → 웅진도독부 설치된 사비 포위 → 임존성으로 근거지 이동 → 복신이 도침 죽이고 실권장악 → 부여풍 암살시도 발각되어 처형 → 내분으로 약화
- **백강 전투** - 부여풍 요청으로 왜국 군사도 전멸, 백제부흥운동 마지막 전투

### 고구려 멸망과 부흥운동
- **안시성 전투** - 연개소문이 영류왕 죽이고 보장왕 세움 → 당나라 침공 → 고구려가 막아냄
- **고구려 멸망** - **함원전**, 평양에 안동도호부 설치  
- **고구려부흥운동** - 검모잠이 안승을 보덕국왕으로 추대, 흑치상지·임존산·소정방 관련

### 나당전쟁
- **기벌포 전투** - 신라가 당군 격파
- **매소성 전투** - 문무왕 시기
- **연정토** - 고구려 대신, 신라에 항복

## 🔄 **주요 사건 순서**

### **서경천도 관련**
인종때: 이자겸·척준경의 난 → 김부식이 이자겸의 난 진압 → 서경천도운동(묘청·정지상/서경파) → 묘청의 난 → 김부식이 묘청의 난 제압 → 삼국사기 편찬(김부식 주도)

### **무신정권 순서**
의종때 정중부·이의방 무신정변(1170) → 조위총의 난(정중부 무신정권 반발) → 김보당의 난(의종 복위 시도) → 이의민이 의종 시해(자객) → 경대승(정중부 제거 시도) → 이의민 집권 → 최충헌(이의민 제거/교정도감) → 최충헌이 명종에게 봉사10조 → 만적의 난(최충헌 노비) → 최우(정방/강화천도) → 최의 → 임유무 → 무신정권 붕괴 → 삼별초 항쟁

### **대외관계 순서**
거란(만부교→광군→서희→강조정변→양규→강감찬) → 여진(윤관 별무반→동북9성→금 사대) → 이자겸의 난 → 서경천도운동(반금정책) → 몽골(저고여→강화천도→처인성→충주성→개경환도) → 원간섭기

### **거란 침입 상세**
만부교 사건(태조/거란 사신단 죽임) → 정종 광군 창설 → 1차(소손녕 침입/서희 담판/강동6주) → 강조의 정변(목종 살해) → 2차(거란 침입 구실/현종 나주피난/양규 흥화진) → 3차(강감찬 귀주대첩) → 현종 나성 축조/초조대장경

### **몽골 항쟁 상세**
사신 저고여 피살 → 최우 강화천도 → 처인성(김윤후/살리타 사살/2차) → 충주성(김윤후/5차) → 귀주성(박서) → 죽주성(송문주) → 몽골 항복/개경환도 → 삼별초 항쟁(배중손)

### **중요 연대순**
1. 태조 왕건 → 정종(왕규의 난) → 광종(과거제) → 경종(전시과) → 성종(12목) → 목종(강조의 정변)
2. 인종(이자겸의 난/서경천도/묘청의 난) → 의종(무신정변) → 명종(최충헌) → 고종(몽골침입)
3. 정중부 무신정권 → 조위총·김보당의 난 → 이의민 → 최충헌 → 최우 → 최의 → 임유무
4. 강조의 정변(목종/문벌귀족 이전) → 이자겸의 난(인종/문벌귀족) → 무신정변(의종/무신정권)
5. 초조대장경(거란/현종) → 재조대장경(몽골) → 원간섭기(개경환도 후)
6. 서희 강동6주 → 윤관 동북9성 → 쌍성총관부(원) → 공민왕 쌍성총관부 공격
7. 광종 과거제(쌍기) → 성종 국자감 → 최충 9재학당 → 예종 7재 → 공민왕 성균관

## 📌 **핵심 연결고리**

### **왕조 관련**
- 왕규의 난(정종 즉위 후) ↔ 강조의 정변(목종) 사이: 성종 12목 설치
- 삼국시대 보장왕(고구려 마지막왕) ↔ 고려시대 살리타(몽골 장군)

### **전투 관련**
- 고창전투(견훤 격파) → 경순왕(김부) 귀순 → 일리천 전투(신검 격파)

### **군사 제도**
- 광군(거란대비/정종) → 별무반(여진대비/숙종/윤관) → 삼별초(대몽항쟁/최우)
- 삼별초 = 최씨 무신정권의 군사적 기반 = 좌별초·우별초·신의군
- 별무반 = 신기군·신보군·항마군(여진 정벌)

### **문화 교류**
- 만권당(충선왕/원 수도 연경) → 이제현(원 문인과 교류)

### **토지 제도**
- 전민변정도감(공민왕) → 신돈 중심 전민변정사업
- 과전법(여말선초) → 조준 건의 → 신진사대부 경제기반 → 경기 지역 한정

### **무신정변 연결**
- 김부식 아들 김돈중 → 정중부 수염 태움 → 무신정변(1170)
- 최씨 무신정권(최충헌→최우→최의) 붕괴 → 임유무 → 삼별초 항쟁

### **행정 기구**
- 재추양부 = 재신(중서문하성) + 추밀(중추원) → 도병마사 구성원
- 대간 = 어사대 + 낭사(중서문하성)
- 서경권 행사 = 어사대 관원 + 중서문하성 낭사 관원
- 안찰사·호부 = 고려 지방행정

### **기타 연결**
- 김흠돌의 난 = 신문왕(신라)
- 부경 = 고구려 창고
- 동시전 = 신라 지증왕(동시 시장 관리 관청) / 경시서 = 고려~조선초(수도 시전 감독)
- 서긍 = 송 사절/선화봉사고려도경(고려도경)/예성항·개경
- 초조대장경 = 거란 침입(현종/나주 피난 시기) / 재조대장경 = 몽골 침입
- 저고여 피살 = 몽골 사신/쌍성총관부 설치 이전
- 서희 강동6주 = 거란 1차 침입/쌍성총관부 설치 이전
- 원간섭기 = 개경환도 이후 시작
- 삼별초 항쟁 = 개경환도 결정에 반발
- 우별초 = 김윤후 지휘(별도 부대)
- 용장성 = 진도(배중손 지휘)
- 의천 = 숙종 동생/천태종(해동천태종) 개창/해동통보 건의
- 지눌 = 국사(불교)
- 문헌공도 = 최충 호(9재학당)
- 이규보 = 동명왕편 저자/동국이상국집/최충헌 무신정권때
- 정지상 = 관음포 해전(여말선초/왜구)
- 김부식 = 숙종대 과거 합격/이자겸의 난 진압/묘청의 난 제압/삼국사기 편찬 총괄
