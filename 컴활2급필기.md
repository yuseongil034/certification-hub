# 📊 컴활 2급 실기 핵심 요약

---

## 시나리오
변경셀 · 결과셀 찾기 → **이름정의 먼저** → 시나리오 추가

- 이름정의 안 하면 시나리오 요약 보고서에 셀 주소로 표시됨 (이름으로 안 나옴)
- 시나리오 요약 보고서 생성 시 결과셀 범위 정확히 지정할 것
- 변경셀은 여러 개 지정 가능 (Ctrl 클릭)

---

## 데이터 표
결과값 셀에 **=참조 입력 먼저** → 영역 잡기 → 데이터 표 실행

- 행 입력셀 / 열 입력셀 헷갈리지 말 것
  → 표 상단(가로)에 값이 있으면 행 입력셀, 왼쪽(세로)에 값이 있으면 열 입력셀
- 영역은 =참조 셀 포함해서 잡을 것

---

## 정렬
조건 **순서대로** 추가 (위쪽이 우선순위 높음)

- ⚠️ 부분합 사용할 거면 **반드시 정렬 먼저**
- 사용자 지정 정렬 시 기준 열 · 정렬 기준 · 순서 세 가지 모두 확인

---

## 부분합
**기준 열 정렬 먼저** → 전체 영역 잘 잡기 → 부분합 실행

- 부분합 중첩 시 두 번째부터 **"현재 부분합 대치" 체크 해제**
- 윤곽 기호(1·2·3)로 요약 레벨 조절 가능한 거 알아두기
- 제거할 땐 부분합 창에서 **"모두 제거"** 클릭

---

## 통합
영역 잡기 → 입력할 표 & 참조 표 **형태 동일**해야 함

- 형태 다를 경우 → **레이블(행/열 제목) 기준으로 통합 옵션 체크**
- 그래도 다르면 영역을 맞게 조정할 것
- 원본 데이터 연결 체크 시 → 원본 변경되면 통합 결과도 자동 반영됨
- 참조 범위 추가할 때 **"추가" 버튼** 눌러야 목록에 등록되는 거 주의
# 컴활 2급 실기 핵심 요약 - 매크로 & 차트

## 매크로
- 매크로 기록 시작 후 **다른 셀 먼저 클릭**하고 행동할 것
  → 셀 선택 위치까지 매크로에 기록되기 때문
- 글(데이터)은 **미리 입력 후** 매크로 적용할 것
  → 원하는 서식/모양으로 적용하기 쉬움
- 매크로 저장 위치 **현재 통합 문서**로 설정할 것
  → 다른 곳에 저장하면 해당 파일에서 불러와지지 않음
- 단추(버튼)에 매크로 연결 시 **양식 컨트롤** 사용할 것
  → ActiveX 컨트롤과 혼동 주의
- 매크로 실행 후 결과 확인 필수
  → 잘못 기록됐으면 삭제 후 다시 기록

## 차트
- 데이터 범위 수정 · 추가 시 범위 잘못 잡지 않도록 **조심**
- 특정 요소만 선택 · 수정하려면 **차트 더블클릭 → 편집모드 진입** 후 요소 클릭
  → 차트 전체 클릭(1번) vs 편집모드(2번) 구분할 것
- 축 제목, 범례, 데이터 레이블 등 **각 요소는 개별 클릭 후 서식** 적용
- 차트 이동 시 **개체로 이동 vs 새 시트로 이동** 구분할 것
  → 문제에서 시트 지정하면 반드시 확인
- 보조 축 필요한 경우 해당 계열 클릭 → **데이터 계열 서식 → 보조 축** 선택
